3. **构建第二问主分析数据管线**
方法/思路：从样例数据直接取字段，不依赖第一问导出的中间表；对连续变量做训练内 `D2` 风格 IQR 缩尾，再做中位数插补。  
关键决策：缩尾阈值在训练数据内部拟合，再应用到验证/测试；不做全样本先缩尾。  
原因：避免信息泄漏，保证阈值、模型性能和风险分层都是真正训练内确定。  
结果路径：实现写在 [run_q2_analysis.py](C:\Users\lenovo\Desktop\整合分析2\run_q2_analysis.py)；每条路径的数据定义在各自目录下的 `tables/feature_definition.csv` 和 `tables/train_test_split_summary.csv`。

4. **随机森林训练、调参与概率校准**
方法/思路：每条路径都做 `70/30` 分层拆分，训练集内 `5` 折交叉验证调参；原始 RF 概率后再比较 `sigmoid` 和 `isotonic` 两种校准方式。  
关键决策：最终落地时，校准方法只在 `sigmoid/isotonic` 里选，不直接选 `raw`。  
原因：`raw` 用来做对比是合理的，但最终正式输出必须是一个可执行的校准模型；`CalibratedClassifierCV` 本身不能接受 `raw` 作为 method。  
结果路径：

- 校准比较：`tables/calibration_comparison.csv`

5. **三级风险阈值划分**
方法/思路：先在训练集 OOF 校准概率上定阈值，再到测试集评估。`tlow` 用 ROC 的 Youden 指数最大值，`thigh` 用训练集概率的 `90%` 分位数，必要时回退到 `95%` 分位或最小可行高阈值。  
关键决策：所有阈值只用训练集 OOF 概率确定，测试集只做验证。  
原因：这是第二问最关键的口径之一，保证“三级风险”不是看了测试集后反推出来的。  
结果路径：每条路径都有 `tables/probability_thresholds.csv`、`tables/test_risk_group_summary.csv`、`tables/sample_predictions.csv`。  
主结果数值：
- `path1_rf_calibrated`：AUC `1.0000`，低/中/高事件率 `0 / 0.909 / 1.000`
- `path2_rf_calibrated`：AUC `1.0000`，低/中/高事件率 `0 / 0 / 1.000`
- `path1_rf_no_direct_lipids`：AUC `0.8392`，低/中/高事件率 `0.514 / 0.897 / 0.983`

6. **特征阈值回溯：SHAP 与近似规则树**
方法/思路：主模型训练完成后，分别用 `SHAP` 和一个浅层近似决策树去解释“哪些特征值区间把风险推高”。  
关键决策：两套方法都保留。  
原因：SHAP 更贴合随机森林本体，规则树更利于论文里写“若满足 A 且 B，则风险升高”的句子。  
结果路径：

7. **去直接血脂指标的稳健性分析**
方法/思路：在主模型之外，再做两条“no_direct_lipids”路径，专门验证结论是否仍能成立。  
关键决策：不重写一套新算法，继续沿用相同的 RF + 校准 + 阈值框架，只改特征集。  
原因：这样比较才公平，性能变化才真正能归因于“是否纳入直接血脂指标”。

- 含血脂路径几乎是“当前状态筛查”模型

- 去血脂后最稳定的变量变成 `血尿酸`，`Path2` 比 `Path1` 更稳

9. **痰湿质亚群 CART 规则识别**
方法/思路：对每条路径在亚群内单独建 `DecisionTreeClassifier`，通过 `cost_complexity_pruning_path + 5 折 balanced_accuracy` 选最优 `ccp_alpha`，最后导出全部叶节点和高风险叶节点。  
关键决策：高风险叶节点定义为 `positive_rate >= 0.85` 且 `sample_count >= 15`。  
原因：既保证规则强度，又避免叶节点过小导致不稳定。

10. **痰湿质亚群 Apriori 规则挖掘**
方法/思路：用 `mlxtend` 的 `apriori + association_rules`，把样本转为事务数据，再挖掘“条件组合 -> 高血脂阳性”规则。  
关键决策一：Apriori 采用临床优先 + 数据支持度兼顾的离散化。  
关键决策二：如果主阈值下没有规则，就逐步放宽 `support/confidence/lift`。  
原因：亚群阳性率高，若阈值太严会空；若太松又会失去解释价值，所以采用分级回退。  
最终离散化口径：
- `痰湿质`：`<=58 / 59-61 / >=62`
- 其他 8 个体质：`<20 / 20-34 / >=35`
- `ADL总分`、`IADL总分`：`<20 / 20-29 / >=30`
- `TG`：`<=1.7 / >1.7`
- `TC`：`<=6.2 / >6.2`
- `LDL-C`：`<=3.1 / >3.1`
- `HDL-C`：`<1.04 / 1.04-1.55 / >1.55`
- `血尿酸`：按性别上限生成 `高尿酸=是/否`
- `年龄组/性别/吸烟史`：保留离散条件  
结果路径：
- `subgroup_t5/tables/apriori_transactions_preview.csv`
- `subgroup_t5/tables/apriori_rules_all.csv`
- `subgroup_t5/tables/apriori_rules_filtered.csv`
实际规则特征：
- 含血脂路径：规则量很多，核心围绕 `TG`、`TC`、`高尿酸`
- 去血脂路径：规则明显集中到 `高尿酸_是 + ADL/吸烟/性别/痰湿质分层`
